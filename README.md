## Getting-Started

Follow these steps to get your development environment set up:

1. Fork the repository:

```bash
 git clone https://github.com/ValmirBP/NodeJs-Pokedex.git
```
2. Install dependencies:

```bash
 npm install
```

3. Usage

Once previous  steps  were completed run

```bash
npm  start
```
